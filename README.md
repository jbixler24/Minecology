# Minecology

This mod adds many common mushroom species and the ability to forage and cultivate them in addition to several fantasy elements.

New fungi added:
- Fly agaric
- Lion's Mane
- Oyster mushroom
- Reishi

Fungi soon to be implemented:
- Chanterelle
- Chicken of the Woods
- Destroying Angel
- Honey Mushroom
- Porcini
